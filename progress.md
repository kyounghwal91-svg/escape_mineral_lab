### 타이머

### 하단 바 버튼 정렬
- **힌트 버튼 위치**: 도감 패널(x:936~1256) 우측 끝에 맞춰 우측 정렬 (`x: 1156`)
- **힌트 버튼 높이**: 안전 장비 버튼과 통일 (`32px → 34px`)

### 광물 도트 마커 제거
- 광물 위 실험 완료 표시 점 4개 완전 제거 (도감으로 대체)
- `_updateMineralDots` 메서드 및 `_mineralObjectsById` Map 제거

### 드래그 글로우 효과
- 광물 또는 도구 드래그 시 상호작용 가능한 대상에 흰색 글로우 표시
- 글로우 형태: 동그라미 → **물건 형태 테두리** (스프라이트 복제 + 흰색 틴트 + BlurFilter)
- 거리 threshold: 91px 이내 시 글로우 표시 (alpha: 1.0)
- 드롭 시 글로우 전체 초기화
- **석영↔방해석 글로우 추가**: 둘 중 하나를 드래그하면 상대 광물에도 글로우 표시

## Experiment Popup (ExperimentPopup.js)

## 조흔 실험 (ScratchEffect.js)

- **완전 재작성**: 광물을 드래그하여 조흔판에 직접 긁는 방식으로 변경
- 광물이 드래그 가능 (조흔판 왼쪽에서 시작)
- 조흔판을 광물로 긁으면 크레파스 스타일 선 생성 (5겹 두꺼운 선 + 랜덤 오프셋)
- 조흔판 밖으로 선이 나오지 않도록 마스크 적용
- **조흔판 색상**: 연한 회색 (`0xe4e4e4`)
- **흰색 조흔**: 순수 흰색 (`0xffffff`, alpha 1.0)으로 명확하게 표시
- **석영(none)**: 문질러도 흔적 없음 (선 그리기 스킵)
- **광물 크기**: `80px → 114px` (1.3배 × 1.1배)
- 20px 이상 긁으면 실험 완료 → 팝업 하단 결과 출력

## 자성 실험 (MagnetEffect.js)

- **완전 재작성**: 클립 7개가 중앙에 다양한 각도로 흩어진 형태
- 광물이 드래그 가능 (왼쪽에서 시작)
- **자철석**: 광물 90px 이내 접근 시 클립이 달라붙는 애니메이션 → 광물 위에 겹쳐져 드래그 추적
- 가까이 가는 클립 모두 순차적으로 달라붙음 (결과 콜백은 최초 1회)
- **클립 위치**: 극좌표 분산 (반경 10~38px 원 안에 랜덤 배치, 광물 영역 내)
- **비자성 광물**: 클립 무반응, 가까이 가면 "자성 없음" 결과 출력
- **클립 크기**: `73px` (1.3배), 이미지: `paper_clip.png`
- **광물 크기**: `104px` (1.3배), zIndex=0으로 클립이 위에 겹침

## Experiment Popup / HardnessComparePopup

### 팝업 프레임
- **상단 모서리 테두리 복원**: 헤더 배경을 1px 안쪽으로 조정해 상단 좌우 라운드 테두리가 다시 보이도록 수정
- `ExperimentPopup.js`, `HardnessComparePopup.js` 모두 동일 방식 적용

### 굳기 비교 실험 형태 변경
- **자동 긁힘 제거**: 진입 즉시 결과가 출력되던 방식 제거
- **상호작용형 실험으로 변경**: 가운데 방해석 고정, 오른쪽 석영을 직접 드래그하여 문질러야 긁힘 발생
- **결과 출력 위치 조정**: 하단 짙은 네이비 결과 패널 안쪽 중심에 결과 문장이 표시되도록 수정

### 굳기 비교 비주얼 및 판정 조정
- **방해석 크기 확대**: 기존 대비 `1.3배`
- **긁힘 표현 변경**: 검은 직선 → 흰색 중심선과 약한 음영을 겹친 패인 자국 형태
- **긁힘 범위 제한**: 긁힘 자국이 방해석 밖으로 보이지 않도록 마스크와 좌표 클램프 적용
- **긁힘 두께/완료 기준 비례 조정**: 방해석 확대 비율에 맞춰 스크래치 두께와 완료 누적 거리도 함께 확대
- **석영 드래그 스케일 복원 버그 수정**: 드래그 후 석영 이미지가 커진 채 남는 문제 해결
