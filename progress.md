## Lab Scene

### 네비게이션
- **안전 장비 착용하기 버튼 추가**: 장비를 모두 착용하지 않은 경우 하단 중앙에 보라색 버튼 표시. 클릭 시 Equipment Scene으로 이동 (statusManager, mineralManager, safetySystem 상태 유지)

### 타이머
- **타이머 위치 변경**: 오른쪽 하단 → 오른쪽 상단 (`y: H-55 → y: 16`)

### 하단 바 버튼 정렬
- **힌트 버튼 위치**: 도감 패널(x:936~1256) 우측 끝에 맞춰 우측 정렬 (`x: 1156`)
- **힌트 버튼 높이**: 안전 장비 버튼과 통일 (`32px → 34px`)

### 광물 도트 마커 제거
- 광물 위 실험 완료 표시 점 4개 완전 제거 (도감으로 대체)
- `_updateMineralDots` 메서드 및 `_mineralObjectsById` Map 제거

### 드래그 글로우 효과
- 광물 또는 도구 드래그 시 상호작용 가능한 대상에 흰색 글로우 표시
- 글로우 형태: 동그라미 → **물건 형태 테두리** (스프라이트 복제 + 흰색 틴트 + BlurFilter)
- 거리 threshold: 91px 이내 시 글로우 표시 (alpha: 1.0)
- 드롭 시 글로우 전체 초기화
- **석영↔방해석 글로우 추가**: 둘 중 하나를 드래그하면 상대 광물에도 글로우 표시

## Experiment Popup (ExperimentPopup.js)

- **내부 패널 색상**: `0x111926` (어두운 남색) → `0xfde8cc` (연한 주황색)

## 조흔 실험 (ScratchEffect.js)

- **완전 재작성**: 광물을 드래그하여 조흔판에 직접 긁는 방식으로 변경
- 광물이 드래그 가능 (조흔판 왼쪽에서 시작)
- 조흔판을 광물로 긁으면 크레파스 스타일 선 생성 (5겹 두꺼운 선 + 랜덤 오프셋)
- 조흔판 밖으로 선이 나오지 않도록 마스크 적용
- **조흔판 색상**: 연한 회색 (`0xe4e4e4`)
- **흰색 조흔**: 순수 흰색 (`0xffffff`, alpha 1.0)으로 명확하게 표시
- **석영(none)**: 문질러도 흔적 없음 (선 그리기 스킵)
- **광물 크기**: `80px → 114px` (1.3배 × 1.1배)
- 20px 이상 긁으면 실험 완료 → 팝업 하단 결과 출력

## 자성 실험 (MagnetEffect.js)

- **완전 재작성**: 클립 7개가 중앙에 다양한 각도로 흩어진 형태
- 광물이 드래그 가능 (왼쪽에서 시작)
- **자철석**: 광물 90px 이내 접근 시 클립이 달라붙는 애니메이션 → 광물 위에 겹쳐져 드래그 추적
- 가까이 가는 클립 모두 순차적으로 달라붙음 (결과 콜백은 최초 1회)
- **클립 위치**: 극좌표 분산 (반경 10~38px 원 안에 랜덤 배치, 광물 영역 내)
- **비자성 광물**: 클립 무반응, 가까이 가면 "자성 없음" 결과 출력
- **클립 크기**: `73px` (1.3배), 이미지: `paper_clip.png`
- **광물 크기**: `104px` (1.3배), zIndex=0으로 클립이 위에 겹침

## 추가 수정 내역 (최근 반영)

### Door Scene
- **단서 영역 구조 개선**: 기존의 단일 목록 형태에서 Lab Scene과 동일한 **3개 광물별 힌트 패널** 구조로 변경
- **상태 동기화**: `MineralManager`의 `getHintPanelState()`를 연동하여 Lab Scene에서 해금된 단계(1, 2단계)까지만 정확히 표시
- **비주얼**: 패널별 번호 배지 및 해금 상태에 따른 색상 구분(녹색/청색) 적용

### Result Scene
- **배경 이미지 적용**: 게임 결과에 따른 전용 배경 이미지 추가
    - 완전 성공: `result_sucess.png`
    - 겨우 성공: `result_barely.png`
    - 실패: `result_failed.png`
- **레이아웃 왼쪽 정렬**: 배경 이미지를 가리지 않도록 모든 텍스트(타이틀, 부제), 통계 박스, 다시 시작 버튼을 **화면 왼쪽(`x: 120`)**으로 배치
- **가독성**: 배경 위 텍스트 시인성 확보를 위해 타이틀 및 부제에 드롭 쉐도우 효과 추가

### Lab Scene
- **힌트 패널 크기 조정**: 상단 단서 힌트 카드의 세로 길이를 **5px 축소 (86px → 81px)**
- **정렬 최적화**: 축소된 높이에 맞춰 '힌트 잠김' 등 내부 요소의 수직 중앙 정렬 위치 보정

### UI/UX 안정성 (Clamping)
- **드래그 범위 제한**: 실험 팝업 및 경도 비교 팝업에서 광물/도구가 화면 밖으로 나가지 않도록 드래그 범위를 제한
    - `HardnessComparePopup`: 석영 광물이 팝업 박스 내부에서만 움직이도록 제한
    - `Acid/Magnet/Scratch Effects`: 각 실험 도구/광물이 500x280 크기의 트레이 영역 내부로 클램핑 처리
