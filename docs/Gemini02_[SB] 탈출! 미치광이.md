# [SB] 탈출! 미치광이 과학자의 광물 실험실

## 0. 전역 설정 (Global Settings)
- **플랫폼:** Web (PC/태블릿 대응, 가로 모드 최적화)
- **아트 컨셉:** 2D 카툰 스타일 (굵은 외곽선, 비비드한 색감)
- **제한 시간:** 10:00 (카운트다운)
- **체력(HP):** 100 (안전 사고 시 차감)
- **사운드:** 과학자 대사는 타이핑 효과음 처리, 실험 도구별 특화 SFX 적용
