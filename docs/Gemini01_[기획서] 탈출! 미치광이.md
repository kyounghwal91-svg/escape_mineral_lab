# [최종 확정 기획서] 탈출! 미치광이 과학자의 광물 실험실

## 1. 프로젝트 개요
- **장르:** 교육용 시뮬레이션 방탈출 웹앱 (2D 기반)
- **컨셉:** 미치광이 과학자의 실험실에서 광물의 특성을 파악해 '열쇠 광물'을 찾아 탈출함.
- **핵심 목표:** 광물 식별 능력 배양 및 실험 안전 수칙 체득.

---

## 2. 기술 스택 및 개발 환경
- **플랫폼:** Web (PC / Tablet 대응)
- **개발 방식:** 2D 엔진 기반 (Canvas API / PixiJS 활용)
- **물리 구현:** - **스크래치:** 드래그 궤적을 따라 단순 선(Line)이 생성되는 방식.
  - **염산 반응:** 미리 제작된 GIF 또는 Lottie 애니메이션(거품 발생) 오버레이.
- **예외 처리:** 브라우저 종료 시 데이터 저장 없이 '처음부터 다시' 시작.

---

## 3. 게임 시스템 및 규칙
### 3.1 핵심 로직 (열쇠 생성)
- **랜덤 생성:** 5종 광물 중 3종을 열쇠로 선정.
- **필수 페어:** [석영-방해석] 중 1종, [흑운모-자철석] 중 1종을 반드시 포함하여 실험 유도.

### 3.2 시간 및 체력 (HP)
- **제한 시간:** 10분 (600초) 고정 타이머.
- **체력(HP):** 100 시작. 0 도달 시 게임 오버.
  - 안전 장비(보안경, 장갑, 실험복) 미착용 시: 실험 시도 시마다 -20 HP.
  - 묽은 염산 취급 부주의(피부 접촉 판정): -30 HP.
  - 유리 기구 파손/무리한 긁기: -15 HP.

### 3.3 UI/UX 상세
- **자동 도감:** 실험(조흔색, 자성 등)이 완료될 때마다 도감에 "자철석: 자성 있음" 등이 자동으로 체크됨.
- **초기화 버튼:** 광물 확대 화면 내 '세척/초기화' 버튼을 통해 염산 흔적이나 긁힘을 초기 상태로 되돌림.

## 5. 엔딩 시나리오 (Ending)

1. **[완벽한 탈출]**
   - **조건:** 10분 내 성공 + HP 100 유지
   - **연출:** 과학자가 분해하며 문을 열어줌. "완벽한 안전 수칙이라니... 할 말이 없군!"
2. **[겨우겨우 탈출]**
   - **조건:** 10분 내 성공 + HP 1~99 사이
   - **연출:** 과학자가 비웃음. "탈출은 축하하네만, 몸 상태가 엉망이군. 다음엔 좀 더 조심하게나!"
3. **[실험 실패]**
   - **조건:** 시간 초과 혹은 HP 0 도달
   - **연출:** 화면이 암전되며 과학자의 광기 어린 텍스트 노출. [다시 도전] 버튼 활성화.

## 6. 리소스 가이드
- **아트:** 굵은 외곽선의 카툰 스타일. 광물은 특징이 뚜렷하게 강조된 일러스트 활용.
- **사운드:** - 과학자 대사는 음성 없이 **타이핑 효과음**과 함께 텍스트로만 처리.
  - 광물 특성별(석영/방해석 등) 긁히는 소리의 질감을 다르게 제작하여 청각적 단서 제공.
